# Protocols — BrandOS Infrastructure

## 用途
此目錄存放 BrandOS 的核心協議文件，定義系統運作的標準程序與互動規則。

## 協議文件規劃

- `ai-infrastructure-protocol.md` — AI 基礎設施核心協議（原 ai-infrastructure-protocol，現整合進 BrandOS）
- `brand-governance-protocol.md` — 品牌治理協議
- `geo-optimization-protocol.md` — GEO 優化標準協議
- `data-privacy-protocol.md` — 數據隱私與安全協議
- `agent-coordination-protocol.md` — 多代理協作協議

## 核心理念
BrandOS Infrastructure = ai-infrastructure-protocol 的正式執行框架
所有協議遵循 WHY → WHAT → HOW 結構。
