# Daily Briefing — 每日情境模板
## commands/daily-briefing.md v1.0 | Symcio × BrandOS

---

## 用途（WHY）

每次開啟新的 Claude.ai Project 對話時，用此模板說明當日工作情境，讓 Claude 快速進入狀態，不需重複解釋背景。

---

## 使用方法（HOW）

複製以下模板，填入當日資訊後貼到對話開頭。

---

## 模板

```
【今日情境】
日期：YYYY-MM-DD
工作者：Sall（Symcio CEO）

【當前任務】
客戶：[客戶品牌名稱 或 Symcio 內部]
任務類型：[品牌分析 / ESG報告 / GEO策略 / 商業提案 / 文案撰寫 / 系統規劃]
任務描述：[2-3句說明今天要完成什麼]

【相關背景】
- 進行中專案：[專案名稱 + 當前 Stage]
- 關鍵限制：[時間限制 / 格式要求 / 特殊注意事項]
- 參考資料：[上次對話結論 / 客戶資料 / 報告連結]

【預期產出】
- 格式：[文件 / 簡報 / 程式碼 / 資料庫記錄 / 摘要]
- 長度：[簡短 / 標準 / 完整]
- 對象：[內部使用 / 客戶簡報 / 公開發布]

【優先順序】
今天最重要的一件事：[填入]
```
